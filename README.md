### flutter 프로젝트 생성
1. cmd창에서 프로젝트를 만들고 싶은 곳으로 이동  
   ex) cd Documents

2. flutter 프로젝트 이름을 넣어 생성 명령어 입력   
   ex) flutter create dowum

3. 프로젝트 폴더로 이동   
   ex) cd dowum

4. vs 코드 실행   
   ex) code .
